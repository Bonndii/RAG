# PDF-RAG-Assistant

RAG-приложение для диалоговой работы с PDF-документами.

## Что делает

* Позволяет загрузить PDF через веб-интерфейс Chainlit
* Извлекает текст из PDF и разбивает его на чанки
* Создаёт embeddings для чанков документа
* Сохраняет векторы и метаданные в локально развернутую векторную БД Qdrant
* Позволяет выбрать уже существующую коллекцию Qdrant
* Поддерживает диалоговый режим с учётом истории сообщений
* Переписывает follow-up вопросы в самостоятельные поисковые запросы
* Возвращает ответ модели вместе с найденными источниками

## Модели

| Задача | Модель |
|---|---|
| Embeddings / семантический поиск | [BAAI/bge-m3](https://huggingface.co/BAAI/bge-m3) |
| LLM для генерации ответа | Чат-модель, доступная на OpenRouter (по умолчанию xiaomi/mimo-v2.5-pro) |

Модель для семантического поиска работает локально.
